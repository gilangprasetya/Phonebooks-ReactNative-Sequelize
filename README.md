# A React Native app running on iOS.

#BACK-END  
- Using Sequalize (Posgresql).

#FRONT-END
- Using React Native

## Project setup on server (ORM)
```
npm run dev
```

## Project setup on client (ClientPhonebooks)
```
yarn start
yarn ios
```
